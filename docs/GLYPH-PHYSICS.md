# GLYPH PHYSICS

**Design reference for archetype glyph geometry.**

This document defines the relationship between archetype physics and glyph geometry. Use it when creating glyph SVGs that follow glyph-law (see `docs/GLYPH-LAW.md`).

---

## Principles

- **Glyph geometry expresses the physics** of the archetype's cosmic analogue and field behavior.
- **Physical behavior class** — how the field resolves (ignition, equilibrium, flow, etc.).
- **Geometric rule** — the symmetry and shape language used in the glyph.
- **Geometry must obey glyph-law constraints**: centered at (500,500), respect inner ring (r=205), do not intersect center dot (r=85), 2–4 primitives, legible at ~40px.

## Archetype Entries

### Ignispectrum

| Field | Value |
|-------|-------|
| **Physical behavior class** | Ignition / Outflow |
| **Cosmic analogue** | Protostar ignition + bipolar jets |
| **Geometric rule** | 3-fold rotational symmetry (120°); outward-pointing elements |
| **Geometry expresses physics** | Three identical elements rotated at 0°, 120°, 240° mimic collimated outflow along multiple axes. The triangles point radially outward from center, echoing core ignition and directional energy shear. Bipolar symmetry becomes 3-fold in the glyph plane. |

## Summary: Physical Behavior → Geometric Rules

| Physical behavior | Typical geometric rule |
|-------------------|------------------------|
| Ignition / Outflow | 3-fold rotational; outward-pointing |
| Equilibrium / Stability | High symmetry; horizontal; centered |
| Replication / Paired | Bilateral; mirror; twin elements |
| Absorption / Indirect | Concentric; occluded; depth |
| Radiance / Central | Radial; outward from center |
| Precision / Periodic | Grid; sharp edges; regular intervals |
| Three-body balance | Bilateral; triangular; counterweights |
| Occlusion / Silhouette | Asymmetry; layered; rim definition |
| Direction / Collimated | Single axis; diagonal; linear |
| Filamentary / Network | Grid; modular; skeletal |
| Shock / Transition | Novel forms; sharp boundary |
| Flow / Spiral | Curved; laminar; rotational |
